# 🚀 Desafio Git - Prática

Treinamento interno na empresa Minsait  
Bootcamp Automação Selenium 2026 - Módulo Git na prática

## 🎯 Objetivo

Aplicar na prática os principais conceitos de Git, simulando um fluxo real de trabalho em equipe.

## 🧪 Cenário real simulado

Este projeto simula um fluxo de trabalho comum em times de desenvolvimento, onde múltiplas pessoas alteram o mesmo arquivo simultaneamente, gerando conflitos que precisam ser resolvidos manualmente.

## 🔄 Fluxo realizado

- Inicialização de repositório Git
- Versionamento de arquivos
- Criação de branch (`feature/conflito`)
- Alterações em branches diferentes
- Merge entre branches
- Simulação de conflito
- Resolução manual de conflito
- Atualização do repositório remoto

## ⚠️ Conflito de merge

Foi gerado um conflito ao editar o mesmo arquivo (`arquivo.txt`) em duas branches diferentes.

A resolução foi feita manualmente, garantindo que ambas as alterações fossem consideradas no resultado final.
