## 🔥 Simulação de Conflito

Foi realizado um cenário de conflito ao editar o mesmo arquivo (`arquivo.txt`) em duas branches diferentes (`main` e `feature/conflito`).
Durante o merge, o Git identificou o conflito automaticamente.

A resolução foi feita manualmente, garantindo a integridade das informações de ambas as alterações.
